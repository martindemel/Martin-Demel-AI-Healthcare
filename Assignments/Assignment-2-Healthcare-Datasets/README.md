# Assignment 2: Healthcare Datasets

## Overview

This assignment practices working with basic healthcare data by reviewing a mock outpatient clinic dataset and calculating simple descriptive statistics.

## Approach

A provided dataset of five patient visit records was analyzed. Each record included a Patient ID, Age (years), and Heart Rate (bpm). The average (arithmetic mean) was calculated for both numerical variables using the formula: Average = sum of values / number of observations.

## Results

| Patient ID | Age (years) | Heart Rate (bpm) |
|------------|-------------|-------------------|
| 1001 | 45 | 72 |
| 1002 | 62 | 78 |
| 1003 | 28 | 88 |
| 1004 | 55 | 76 |
| 1005 | 39 | 70 |

| Variable | Average |
|----------|---------|
| Age | 45.8 years |
| Heart Rate | 76.8 bpm |

## Key Takeaway

Basic summary statistics derived from healthcare visit data can support simple insights about a patient group. This exercise demonstrates the foundational skill of working with structured health data, which scales to larger datasets used in AI model development.
